代表からのメッセージ

考える力と創る情熱を持つ仲間を募集しています！

これまでの人生を振り返ると、自分の性格や特性から、社会に馴染むことが難しいと感じる場面が多々ありました。

たとえば、不器用でいまだにブラインドタッチがうまくできなかったり、新しいツールを使いこなすのに時間がかかることもあります。 また、人見知りな性格のせいで、あいさつをするたびに緊張して心臓がドキドキしてしまうこともあります。 しかし、そんな自分だからこそ、同じように悩みや不安を抱える方々が、自分らしく安心して過ごせる環境をつくりたいという思いを持つようになりました。

私たちの会社では、一人ひとりの個性や背景を尊重し、それぞれが持つ特性や経験が新しい価値を生み出す力になると信じています。 完璧である必要はありません。 大切なのは、互いに支え合いながら一歩ずつ前へ進むことです。

もしあなたが「生きづらさ」を感じているなら、それはあなた自身の可能性がまだ十分に発揮されていないだけかもしれません。 その可能性を広げ、新しい未来へ踏み出すお手伝いを、私たちは全力でサポートしたいと思っています。 あなたとの出会いを心から楽しみにしています。

私たちが大切にしていること

私たちは、お客様のビジネスに並走し、プロジェクト全体をサポートするパートナーです。年次や経験にかかわらず、設計や要件定義といった上流工程にもチャレンジできる環境で、共に成長し、お客様の成功に貢献してくれる仲間を募集しています。

募集要項

仕事内容

経験・スキルに応じて、Web系・業務系システムの開発をお任せします。 お客様のビジネスを深く理解し、プロジェクトを成功に導くパートナーとして、PMやチームメンバーと連携しながら機能追加や改善の提案・実装を行います。

  • Web系・業務系システムの開発業務 (PHP/JavaScript等)
  • 基本設計・詳細設計の作成、仕様策定
  • コードの品質向上、テスト・デバッグの実施

主要業界・領域

私たちは、特に以下の業界におけるお客様のDX推進を支援しています。

  • 建設業界 (施工管理、業務支援システム)
  • 中古車流通業界 (在庫管理システム、売買支援ツール)
  • 製造・物流業界 (生産管理、倉庫管理システム)

主な開発環境

  • 言語・フレームワーク: PHP (Laravel), Vue.js, Next.js, React
  • データベース: MySQL, PostgreSQL
  • インフラ・クラウド: AWS (EC2, S3, Lambda, RDS)
  • 開発ツール: GitHub, Docker, JIRA, Slack

求めるスキル・経験

  • Webオープン系システムの開発経験 (1年以上)
  • 設計・要件定義の経験がある方 (システムエンジニア志向の方歓迎)
  • フロントエンド (Vue.js, Reactなど) またはバックエンド (PHPなど)の実装経験
  • AWSなどのクラウド環境での開発経験
